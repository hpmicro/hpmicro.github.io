14.4. lv_demo_stress¶

14.4.1. 概述¶

LVGL的官方压力测试。该示例程序包含多种对象创建、删除、动画、风格使用等。该示例程序可以用于测试内存泄漏、错误。

更多详细信息请参考官方 lv_demos github

14.4.2. 硬件连接¶

将LCD屏连接到评估板

14.4.3. 运行¶

littglevgl stress将会显示在LCD屏上

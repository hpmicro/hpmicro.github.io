6.29.7. SPI_MASTER_TRANS_LARGE_AMOUNT_OF_DATA¶

6.29.7.1. 概述¶

spi_master_trans_large_amount_of_data示例工程展示了SPI master如何使用hpm_spi component传输大量数据。 在这个示例工程中，SPI Master以双工的方式收发数据(当传输数据数量超出SPI一次传输的上限时，hpm_spi component会使用DMA链的方式将SPI传输链接起来)。

6.29.7.2. 硬件设置¶

将两个板子的SPI引脚MOSI与MISO相连。

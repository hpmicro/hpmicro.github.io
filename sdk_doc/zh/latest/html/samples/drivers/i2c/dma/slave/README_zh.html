6.29.4. i2c_dma_slave¶

6.29.4.1. 概述¶

i2c_dma_slave示例工程展示了I2C在slave模式使用dma方式接收和发送数据的功能。在这个示例工程中，I2C slave先接收数据，之后将接收到的数据发送出去。

6.29.4.2. 硬件设置¶

将两个板子的I2C引脚相连。

6.29.4.3. 运行现象¶

当工程正确运行后，串口终端会输出如下信息：

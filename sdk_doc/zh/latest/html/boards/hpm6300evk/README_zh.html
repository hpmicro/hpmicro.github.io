4. HPM6300EVK开发板¶

4.1. 概述¶

  • 控制台串口参数：

    • 波特率：115200

    • 数据位：8位

    • 校验位：无

    • 停止位：1

    • 流控位：无

    • 数据编码方式：ASCII

4.2. 拨码开关 S1¶

  • Bit 1，2控制启动模式

Bit[2:1]

功能描述

OFF, OFF

Quad SPI NOR flash 启动

OFF, ON

串行启动

ON, OFF

在系统编程

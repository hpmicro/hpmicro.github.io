6.28.3. SPI_DMA_MASTER¶

6.28.3.1. 概述¶

spi_dma_master示例工程展示了SPI作为master使用dma方法进行板与板之间通信。 在这个示例工程中， 一个板子作为SPI master， 另一个板子作为SPI slave。 Master发送一定量数据到Slave并从Slave接收一定量的数据。

6.28.3.2. 硬件设置¶

将两个板子的SPI引脚相连。

6.28.3.3. 运行现象¶

当工程正确运行后，串口终端会输出如下信息：

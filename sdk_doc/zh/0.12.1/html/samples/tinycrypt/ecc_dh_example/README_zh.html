9.7. 差错检测和修正的算法ECC-DH¶

9.7.1. 概述¶

ecc_dh_example示例工程展示了差错检测和修正的算法ECC的实现。

  • 此工程的差错检测和修正的算法采用ECC DH完成模块加解密测试。

9.7.2. 硬件设置¶

  • 无特殊设置

19.2. RAMSTERSS example¶

19.2.1. 概述¶

RAMSTERSS 示例工程对外部的SDRAM进行压力测试. 主要包含如下测试内容:

  • algo_fill_selfaddress

  • algo_random

  • algo_or

  • algo_xor

  • algo_and

  • algo_sub

  • algo_mul

  • algo_div

  • algo_seqinc

  • algo_solidbits

  • algo_checkerboard

  • algo_blockseq

  • algo_walkbits0

  • algo_walkbits1

  • algo_bitspread

  • algo_bitflip

  • algo_8bits

  • algo_16bits

19.2.2. Board Setting¶

无需特殊设置

19.2.3. Running the example¶

当工程正确运行后，串口终端会输出如下信息：

=============================================
All cases are PASSED
=============================================

*注意: 这个工程大约需要花费 50 分钟.请耐心等待!
